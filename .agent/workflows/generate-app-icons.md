# 批量生成应用图标

使用 generate_image 批量生成应用所需的图标。

## 当前项目图标需求

### 1. 关卡图标 (25个)

| 关卡 | 主题 | 描述 |
|------|------|------|
| 1-5 | CLI 基础 | 终端、命令行 |
| 6-10 | 参数控制 | 开关、滑块 |
| 11-17 | 斜杠命令 | 闪电、快捷 |
| 18-25 | 高级功能 | 齿轮、火箭 |

生成示例：
```
生成关卡图标：
- 风格：扁平化、圆形背景
- 尺寸：64x64 PNG，透明背景
- 主色：Indigo (#6366f1)
- 数量：5个一组
- 主题：终端/命令行
```

### 2. 成就徽章

| 成就 | 描述 |
|------|------|
| 初学者 | 完成第一个关卡 |
| 探索者 | 完成 10 个关卡 |
| 精通者 | 完成全部关卡 |
| 速通达人 | 30分钟内完成教程 |
| 完美主义 | 无错误完成 |

生成示例：
```
生成成就徽章：
- 风格：游戏化、金属质感
- 尺寸：128x128 PNG
- 边框：金色/银色/铜色
- 中心图案：星星/奖杯/皇冠
```

### 3. 应用图标

```
生成应用图标：
- 用途：favicon、PWA 图标
- 尺寸：16x16, 32x32, 192x192, 512x512
- 设计：终端光标 + Indigo 背景
```

## 图标保存位置

```
web-game-vue/
├── public/
│   ├── favicon.ico
│   └── icons/
│       ├── icon-192.png
│       └── icon-512.png
└── src/
    └── assets/
        └── images/
            ├── levels/
            │   ├── level-1.png
            │   └── ...
            └── badges/
                ├── beginner.png
                └── ...
```

## 使用生成的图标

```vue
<!-- 关卡图标 -->
<img :src="`/assets/images/levels/level-${level.id}.png`" />

<!-- 成就徽章 -->
<img :src="`/assets/images/badges/${badge.key}.png`" />
```
