# 生成 UI 素材

使用 `generate_image` 工具生成项目所需的图片素材。

## 使用方法

在对话中告诉我需要什么类型的素材，例如：

### 1. Hero 背景
```
生成首页 Hero 背景：渐变紫色调，带有抽象几何图形，现代科技感
```

### 2. 关卡图标
```
生成关卡图标：终端/命令行主题，扁平化设计，尺寸 64x64
```

### 3. 成就徽章
```
生成成就徽章：金色边框，中心有星星图案，游戏化风格
```

### 4. 空状态插图
```
生成空状态插图：友好的机器人角色，"暂无数据"场景
```

### 5. 教程步骤图
```
生成教程步骤图：标注箭头，高亮区域，引导用户操作
```

## 素材风格指南

- **主色调**: Indigo (#6366f1)
- **风格**: Clean Minimal / Modern Tech
- **格式**: PNG (透明背景) 或 JPG
- **尺寸**: 根据用途指定

## 生成后操作

1. 素材会保存到 artifacts 目录
2. 复制到 `web-game-vue/src/assets/images/`
3. 在组件中引用使用
